# Product sizing guide

Footwear: Use US sizing; wide sizes noted with W suffix. If between sizes for running shoes, choose half-size up for toe room.

Apparel: Jackets are standard fit; size up for layering. Refer to chest/waist measurements on each PDP.

Backpacks: Lumbar capacity is listed in liters; hip belts are adjustable for 28–40 inch waists.
