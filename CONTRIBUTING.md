# Contributing

## Quick setup

```bash
git clone <repo-url>
cd chatgptapp-mcp
cp .env.example .env
docker compose up --build -d
```

## Pull requests

- Keep changes focused and consistent with existing style.
- Do not commit `.env` or real AWS keys, tokens, or ngrok secrets.
- Update `docs/` or `README.md` when behavior or setup changes.
